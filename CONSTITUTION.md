# 铺装宪法 (Paving Constitution)

木地板铺装图生成工具的不可违反定则。所有算法、优化、展示均须遵守。

---

## 第一条：榫槽定向 (Tongue-and-Groove Orientation)

```
      母榫 (groove)
    ┌─────────────┐
    │             │
母榫│   地板 L×W  │公榫 (tongue)
    │             │
    └─────────────┘
      公榫 (tongue)
```

- **1.1** 板不可翻转 180° — 翻转后公榫变母榫，无法对接
- **1.2** 连接只能公榫入母榫（tongue → groove），不可公-公或母-母

## 第二条：切割边靠墙 (Cut Edge → Wall)

- **2.1** 切割面无榫槽结构 → 必须靠墙（房间边界）
- **2.2** 长度切割边和宽度切割边均须遵守
- **2.3** 完整板的任意边可靠墙

## 第三条：宽切最多一次 (Single Width-Cut per Board)

- **3.1** 一块板沿宽度方向最多切一次，产出：
  - 1 条使用带（保留一条原始长边 + 一条切割长边）
  - 1 条废料带（保留另一条原始长边 + 一条切割长边）
- **3.2** 废料带保留一条原始边（公榫或母榫）→ 可作边界板复用
- **3.3** 废料带不可再次宽切 — 再切产生的中间带两边皆切割面 → 无法连接 → 不可用
- **3.4** 宽度复用消耗整条宽度，多余部分废弃（双切面）

## 第四条：长切可多次 (Multiple Length-Cuts Allowed)

- **4.1** 沿长度方向可切多次，产生 N 段
- **4.2** 每段保留两条原始长边（公榫 + 母榫），方向不变
- **4.3** 切割短边（截面）靠墙
- **4.4** 长度尾料可入池跨行/跨房间复用

## 第五条：满铺无隙 (Complete Coverage)

- **5.1** 可铺区域（房间 − 踢脚线 − 伸缩缝）必须完全覆盖
- **5.2** 铺装板不得超出房间边界
- **5.3** 板与板之间不留空隙（板间缝/伸缩缝除外）

## 第六条：最少损耗 (Minimum Waste)

- **6.1** 满铺前提下总用板数最少
- **6.2** 利用率 = 房间可铺总面积 / (总用板数 × 单板面积)
- **6.3** 损耗率 = 1 − 利用率

## 第七条：组合切割 (Combined Cut)

- **7.1** 一行末 + 最后一行的板需要同时长切+宽切
- **7.2** 组合切产出两个不重叠的废料区：
  ```
  ┌──────────────┬─────────────────┐
  │ 使用片        │ 宽度废料         │ ← used_len × total_w
  │ used_len×    │                 │
  │ actual_w     │                 │
  ├──────────────┴─────────────────┤
  │ 长度废料（全宽，两原始边完好）    │ ← (L-used_len) × total_w
  └────────────────────────────────┘
  ```
- **7.3** 长度废料入普通池（全宽可用），宽度废料入宽切池（一原始边）

---

## 校验规则 (Verification)

每次运行自动执行 `floorplan/verify.py`：

1. **切割板完整性**：每块源板的 pieces + 废料 + 锯缝 = 整板面积
2. **铺装覆盖率**：铺装板 union 完全覆盖可铺区域，且不超出房间边界

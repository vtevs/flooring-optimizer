# 任务计划 (Task Plan)

## 项目概述

**木地板铺装图生成工具** — CLI 命令行工具，输入 YAML 配置文件
（房间尺寸、地板规格、铺装方式），输出 SVG 铺装图，目标是最小化地板损耗。

**核心能力：**
- 支持 L 形房间（两个正交矩形拼接）
- 支持 4 种铺装方式：对拼、错缝、人字拼(45°)、5拼方砖(棋盘式)
- 支持自定义地板规格（长×宽）、铺装方向、踢脚线宽度
- 支持可配置的伸缩缝（地暖默认 10mm）、错缝比例、板间留缝
- 考虑地板榫槽方向约束（切割板不能翻转复用）
- 输出 SVG 矢量铺装图，含尺寸标注和损耗统计

## 风险与阻塞项

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 多房间排列组合爆炸 | 顺序枚举 N! 随房间数增长 | 限制每次仅 2-3 房间共享池；独立优化作为基线 |
| 排样优化是 NP-hard，最优解计算量大 | 大房间可能耗时长 | 贪心+枚举起始点，不追求数学最优 |
| 榫槽约束使尾料复用计算复杂 | 利用率可能被低估或高估 | 明确记录复用规则，单元测试覆盖 |
| 人字拼 45° 边界裁剪产生很多三角废料 | 损耗率较高 | 枚举起始偏移找最优位置 |
| SVG 大房间图可能文件过大 | 浏览器渲染慢 | 控制精度到 mm，必要时缩放 |
